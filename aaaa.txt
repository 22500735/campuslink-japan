1. 시간표 - 에타 (디자인 변경)

## 1. 시간표 기능

![시간표 옙.png](attachment:e081587c-426d-4a64-a879-3f41f439777b:시간표_옙.png)

- **기본 구조**
- 원형(라운드형) 시간표 UI
- 시간표 위치에 그대로 라운드형 UIO 가 들어오면 좋게씀!
- 월요일~금요일 요일 선택 가능
- 일본 대학 커리큘럼 및 시간표 데이터 기반으로 과목 추가 가능
- **사용자 동작 흐름**
    1. 사용자가 ‘시간표’ 메뉴 클릭
    2. 요일 선택 후, 시간대에 과목 추가
    3. 과목 정보: 강의명, 교수명, 강의실, 학점, 요일/시간
- **추가 기능**
    - **학점 계산기**: 에브리타임과 동일한 방식 유지
    - 추가·수정·삭제 가능

---

## 2. 친구 기능

- **기본 구조**
    - ‘친구’ 메뉴와 ‘시간표’ 메뉴 **분리**
    - 친구 목록 화면에서 친구 이름 클릭 시 해당 친구 시간표 확인 가능
- **사용자 동작 흐름**
    1. ‘친구’ 메뉴 클릭
    2. 친구 목록 표시
    3. 검색창에서 이름 검색 or ‘친구 추가’ 버튼 클릭
    4. 라인 링크를 통해 친구 추가
    5. 친구 클릭 시 해당 친구 시간표 팝업/페이지 전환

---

## 3. 차별화 포인트

1. **메뉴 분리**: 시간표와 친구 페이지를 완전히 분리
2. **친구 시간표 열람 기능**: 친구 목록 → 클릭 → 시간표 확인
3. **일본 로컬 특화**: 라인 링크를 통한 친구 추가, 일본 커리큘럼 데이터 적용

## **4.2 서비스 아키텍처 및 UI 구상**

### ① 전체 서비스 구조

**[1] 클라이언트 레이어 (사용자 인터페이스)**

- **모바일 앱 (iOS/Android)**
    - 일본 대학생 맞춤 UI/UX
    - 다국어 지원(일본어 기본, 한국어/영어 선택 가능)
- **웹 버전**
    - 학사 일정, 강의평, 커뮤니티 열람 가능
    - 관리·운영자 페이지 제공

**[2] 애플리케이션 레이어**

- **학업 모듈**: 시간표, 강의평, 시험·과제 일정
- **커뮤니티 모듈**: 자유게시판, 질문·답변, 동아리 홍보, 중고거래
- **생활 편의 모듈**: 학내 지도, 식단, 행사 일정, 날씨
- **알림/푸시 모듈**: 시험 일정, 커뮤니티 댓글, 이벤트 알림

**[3] 보안·인증 레이어**

- @ac.jp 대학 이메일 인증
- JWT 기반 세션 관리
- 유저 권한 레벨(학생·관리자·모더레이터)
- 콘텐츠 필터링(비속어, 개인정보, 불법 콘텐츠 자동 차단)

**[4] 데이터 레이어**

- 강의·교수 데이터베이스
- 커뮤니티 게시글/댓글 DB
- 사용자 계정/인증 DB
- 이벤트·광고·마켓플레이스 DB


### ④ Penmark 대비 UI 차별화 전략

- Penmark는 **심플·학업 중심 UI** → 우리는 **커뮤니티와 생활 정보의 비중 확대**
- 메인 화면에서 학업+커뮤니티+생활 정보 동시 노출
- 익명 커뮤니티 UX 최적화(작성자 정보 최소 표시, 신고/차단 기능 강화)
- 일본 대학 문화에 맞춘 색상·폰트·아이콘 디자인 (차분한 톤, 정보 위계 명확화)

## **4.3 핵심 기능 상세 설명**

### ① 학업 관리 기능

| 기능 | 세부 설명 | 기술 사양 |
| --- | --- | --- |
| **시간표 생성·관리** | 강의명, 교수명, 강의실, 요일·시간 설정, 드래그&드롭 편집 | React Native 기반 캘린더 UI, SQLite 로컬 저장 + 서버 동기화 |

## **4.4 차별화 전략**

### ① 기능 측면 차별화

| 구분 | Penmark | CampusLink Japan (예정) | 차별화 포인트 |
| --- | --- | --- | --- |
| **학업 관리** | 시간표, 강의평 중심 | 시간표·강의평 + 과제·시험 일정 + 출결관리 | 학업 기능 범위 확장 |
| **커뮤니티** | 제한적, 일부 게시판 | 학과·학년별 + 익명 자유게시판 + 주제별 토론 | 완전 익명, 폐쇄형, 다채로운 주제 |
| **생활 편의** | 거의 없음 | 학내 지도, 학식 메뉴, 교통·날씨, 행사 안내 | 대학 생활 전반 통합 |
| **거래/마켓** | 미지원 | 학내 중고거래, 향후 안전결제 지원 | 커뮤니티 내 경제 활동 활성화 |
| **알림 시스템** | 단순 공지 | 과제·시험·댓글·이벤트 맞춤 알림 | 개인화된 푸시 알림 제공 |

### ② 사용자 경험(UX) 차별화

- **하나의 앱에서 대학 생활 전부 해결**: Penmark는 학업, OpenChat은 대화, SNS는 홍보 → CampusLink Japan은 올인원 구조
- **홈 화면 통합 대시보드**: 시간표 + 최신 게시글 + 학내 이벤트 동시 확인
- **익명 커뮤니티 UX 최적화**: 작성자 정보 최소화, 신고·차단 간편화, 악성 게시물 자동 필터링

---

### ③ 시장 진입 전략 차별화

- **대학별 전담 홍보팀 구성**: 신입생 오리엔테이션, 동아리·학생회 제휴
- **지역별 확산 전략**: 수도권 대형 대학 → 지방 대학 확장
- **입학 시즌 집중 공략**: 4월 신학기 시즌에 맞춘 집중 마케팅
- **초기 보상 프로그램**: 첫 가입자 대상 상품권, 이벤트 포인트 제공

---

### ④ 기술적 차별화

- **보안 수준**: JWT + OTP + AI 기반 콘텐츠 필터링 → Penmark 대비 강화
- **데이터 분석**: 이용자 행동 데이터 기반 맞춤형 콘텐츠 추천
- **확장성 있는 아키텍처**: 학내 정보 외에도 취업, 어학, 유학 정보 연동 가능
